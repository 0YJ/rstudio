Keyboard Shortcuts

Console

Ctrl+1 - put the cursor in the console
Ctrl+L - clear console
Ctrl+A - move cursor to beginning of line
Ctrl+E - move cursor to end of line
Up/Down - navigate command history
Ctrl+Up - popup command history (filtered by command text already entered)
Esc - interrupt currently executing command

Source

Ctrl+2 - put the cursor in the Source document
Ctrl+Shift+N - start new document (except on Chrome for Windows)
Ctrl+O - open document
Ctrl+S - save active document
Ctrl+Shift+L - close active document
Ctrl+Enter (or Ctrl+Alt+L) - run current line/selection
Ctrl+Shift+Enter - Run current document
Ctrl+Option+Down - switch to tab
Ctrl+Option+Right - next tab
Ctrl+Option+Left - previous tab
Ctrl+Shift+Option+Left - first tab
Ctrl+Shift+Option+Right - last tab

Editing (Console & Source)

Ctrl+Z - undo
Ctrl+Shift+Z - redo
Ctrl+X - cut
Ctrl+C - copy
Ctrl+V - paste
Ctrl+U - yank line up to cursor
Ctrl+K - yank line after cursor
Ctrl+Y - insert currently yanked text

Completions (Console & Source)

Tab or Ctrl+Space - attempt completion
Up/Down - navigate candidates
Enter, Tab, Right - accept selected candidate
F1 - show help for selected candidate
Esc - dismiss completion popup

Views

Ctrl+1 - move cursor to source
Ctrl+2 - move cursor to console
Ctrl+3 - show workspace
Ctrl+4 - show data
Ctrl+5 - show history
Ctrl+6 - show files
Ctrl+7 - show plots
Ctrl+8 - show packages
Ctrl+9 - show help

Plots

Ctrl+PgDn - next plot
Ctrl+PgUp - previous plot
